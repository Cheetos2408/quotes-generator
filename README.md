# Quotes Generator

## Описание проекта
Quotes Generator — это веб-приложение для генерации случайных мотивирующих и вдохновляющих цитат. Пользователь видит цитату и автора и может получить новую цитату одним кликом на кнопку.

Приложение ориентировано на минималистичный дизайн и удобный пользовательский интерфейс.



## Функциональность
- Генерация случайной цитаты и автора  
- Кнопка "New quote" для получения новой цитаты  
- Адаптивный дизайн под мобильные устройства  
- Работа с API для получения актуальных цитат  
- Эффект затемнения фона (overlay) для фокусировки на карточке  



## Используемые технологии
- HTML — структура приложения  
- CSS — стили и адаптивная верстка  
- JavaScript — логика приложения, работа с DOM и API  
- API Ninjas — получение случайных цитат  



## Запуск проекта
1. Скачайте проект или клонируйте репозиторий.  
2. Откройте файл `index.html` в браузере.  
3. Нажмите на кнопку ⟳, чтобы сгенерировать новую цитату.  



## Структура проекта
quotes-generator/
│
├── index.html # HTML-разметка приложения
├── style.css # Стили и адаптивная верстка
└── script.js # JavaScript-логика и работа с API




## Работа с API
Приложение использует **API Ninjas** для получения случайных цитат:

- Запрос выполняется через `fetch` с заголовком `"X-Api-Key"`  
- Кнопка "New quote" обновляет цитату и автора  
- API ключ хранится в `API_KEY` и нужен для работы  

Источник API:  
https://api.api-ninjas.com



## Архитектура проекта
Проект построен как клиентское веб-приложение без сервера:

- HTML отвечает за структуру интерфейса  
- CSS отвечает за визуальный стиль  
- JavaScript управляет логикой, DOM и запросами к API  



## Сервер и деплой
- Сервер не используется  
- Приложение является статическим сайтом
